Festix is a ticket marketplace for concerts and music festivals, allowing users to explore events, locations, artists, and ticket prices. The platform also connects users with music communities based on their interests and provides an informative blog for music enthusiasts. In this project, I designed the design system, created wireframes and prototypes in Figma, and developed the frontend using HTML, CSS, JavaScript, and ReactJS. I also built the backend with Node.js and Express.js, implementing features like transactions, authentication, community interactions, and event management. The project was successfully deployed on an Ubuntu server using Nginx..
